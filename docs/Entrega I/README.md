# **Documentación del Proyecto: AI Support Co-Pilot**

**Maestría en Computación para el Desarrollo de Aplicaciones Inteligentes (CODING)** · Universidad del Valle

Este repositorio contiene la documentación requerida para la fase de inicio y planificación del Trabajo Integrador "AI Support Co-Pilot".

---

## Alcance de la documentación

| Tipo | Descripción |
|------|-------------|
| **Semestre I** | Documentos que describen el alcance actual: arquitectura base, auth, CRUD de tickets y clasificación por reglas (sin IA). |
| **Proyecto general** | Visión a largo plazo: integración con IA (LLM), n8n, analítica y automatizaciones en Semestres 2 y 3. |

Algunos artefactos combinan ambos ámbitos para mostrar el recorrido completo del proyecto.
